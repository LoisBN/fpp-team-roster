# Team Roster — Props & Conditional Rendering

Build a team directory with reusable member cards. Practice props, .map(), and learn conditional rendering with boolean values.

## Quick Start

**1. Fork this repo** — Click the **Fork** button at the top right of this page.

**2. Clone your fork:**
```bash
git clone https://github.com/YOUR-GITHUB-USERNAME/fpp-team-roster.git
cd fpp-team-roster
npm install
cp .env.example .env
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the app.
